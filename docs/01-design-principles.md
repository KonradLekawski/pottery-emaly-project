# Design Principles

## 1. Silhouette before detail

If the black side profile does not work, 3D rendering, color and ornament will not save the product.

The first tool must therefore be a silhouette browser, not a CAD editor.

## 2. The 330 ml size is the DNA

The 330 ml form defines the brand. 120, 240 and 440 are adaptations, not linear scales.

## 3. Children are not a separate aesthetic

120 ml and 240 ml are child-adapted versions of the same table language. They should be lower, lighter, safer and more stable, but not infantile.

## 4. Stopka is a brand signature

A flat enamel mug bottom pulls the object toward camping. A subtle foot, undercut, shadow and table presence pull it toward premium tableware.

## 5. The handle is jewelry

The handle is not a loop attached to a body. It is a formal signature. The negative space inside the handle matters as much as the handle body.

For children, the two handles should read as light wings, not training-bottle grips.

## 6. Premium is restraint plus one gesture

Too quiet is dead. Too expressive is fashionable. The form needs one recognizable gesture: foot, rim, waist or handle.

## 7. Family coherence beats perfect single objects

A beautiful 330 ml candidate fails if 120 ml becomes a toy or 440 ml becomes a bucket.

## 8. Geometry must eventually obey production

The current Shape Hunt app is for discovery. Final geometry still needs wall thickness, rim construction, handle attachment, metal-forming feasibility and enamel process validation.
