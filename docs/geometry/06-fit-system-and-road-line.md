# Fit System and Road Line Guardrails

## Why this exists

Shape Hunt should not only generate attractive silhouettes. It should help reject shapes that cannot become a credible family product.

The current fit system adds three design checks:

1. **Immediate 3-model preview** — every gallery card shows 120 / 240 / 330 ml together.
2. **Automatic dimensional filtering** — shapes can be filtered by Table Line proportions and pressed-steel safety.
3. **Road / Cup Holder screening** — saved Table Line DNA can be translated into a separate travel body candidate.

## Table Line dimensional intent

Current visual selection focuses on three primary sizes:

| Size | Handles | Intent |
|---:|---:|---|
| 120 ml | 2 | first table cup, stable, dignified, not toy-like |
| 240 ml | 2 | child bridge size, still part of adult table language |
| 330 ml | 1 | main Table Line DNA |

440 ml remains part of the product platform, but it is not part of the immediate 3-model gallery preview. It should be checked later once the 330 ml DNA is stronger.

## Automatic filters

### All shapes

No dimensional filtering. Useful for broad discovery.

### Table dimension fit

Keeps only candidates whose 120 / 240 / 330 translations stay inside the intended design ranges. This keeps the hunt from drifting into forms that only work as a single 330 ml silhouette.

### Pressed steel safe

Keeps softer, lower-angularity candidates. The objective is to avoid shapes that feel like ceramic fantasy or sharp-kinked renders but would fight metal drawing and enamel coating.

### Road / cup-holder potential

This does **not** mean the Table cup becomes the travel cup. It screens whether the Table DNA could be translated into a separate Road Line:

- narrower base;
- taller, more vertical body;
- lid/closure interface;
- stable cup-holder insertion;
- no side handles that interfere with vehicle holders.

## Road Line assumptions

Road Line is a future product family. It should be treated as its own line, not forced into Table Line geometry.

Current screening assumptions:

| Base diameter | Screening result |
|---:|---|
| ≤ 71 mm | strongest compatibility target |
| 72–77 mm | fits many holders but needs vehicle check |
| 78–83 mm | large / adjustable holders only |
| > 83 mm | reject for broad cup-holder fit |

The closure system is marked as an outsourced module candidate. Shape Hunt only reserves a conceptual closure ring and lid zone; it does not solve gasket, sip mechanism, leakproofness, heat safety or supplier tooling.

## Design principle

The Road Line must inherit Table Line DNA, but it must not deform Table Line into a travel mug.

Table Line remains:

- lower;
- table-first;
- cup/filiżanka-oriented;
- family ritual object.

Road Line becomes:

- vertical;
- cup-holder safe;
- lidded;
- one-hand travel capable;
- more technical, but still warm and recognizable.
