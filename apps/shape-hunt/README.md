# Shape Hunt

A static silhouette-first discovery app for the Pottery Emaly Project.

## Run

```bash
python3 -m http.server 5173
```

Open `http://localhost:5173`.

## Keyboard

- `G` — good
- `B` — bad
- `S` — save
- `M` — more like this
- `←` / `→` — focus previous / next
- `Esc` — close focus

## Purpose

Browse hundreds of silhouettes quickly, build a shortlist, and generate more candidates around promising directions.
